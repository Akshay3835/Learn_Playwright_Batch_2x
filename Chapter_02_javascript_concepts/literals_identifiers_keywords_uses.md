# Use of Literals, Identifiers & Keywords in JavaScript

| Concept | Purpose | Examples |
|---------|---------|----------|
| **Keywords** | Reserved words that have special meaning in the language — used to define syntax, control flow, declarations, etc. | `let`, `const`, `if`, `else`, `for`, `while`, `function`, `return`, `class`, `import` |
| **Identifiers** | Names given to variables, functions, classes, or properties to reference them in code | `myVariable`, `calculateTotal()`, `UserProfile`, `firstName` |
| **Literals** | Fixed values written directly in source code — represent data without needing a variable | `42` (number), `"Hello"` (string), `true` (boolean), `[1, 2, 3]` (array), `{name: "John"}` (object) |

## Summary

- **Keywords** tell JavaScript **what to do** (the language's commands)
- **Identifiers** let you **name things** so you can refer to them later
- **Literals** are the **actual data values** you write in code
