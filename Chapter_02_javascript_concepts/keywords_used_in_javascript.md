# JavaScript Keywords

| Keyword      | Category         | Description |
|--------------|------------------|-------------|
| `break`      | Control Flow     | Terminates a loop or switch |
| `case`       | Control Flow     | Matches a value in switch |
| `catch`      | Exception        | Handles an error |
| `class`      | Declaration      | Declares a class |
| `const`      | Variable         | Declares a block-scoped constant |
| `continue`   | Control Flow     | Skips to next iteration |
| `debugger`   | Debugging        | Invokes debugger |
| `default`    | Control Flow     | Default case in switch |
| `delete`     | Operator         | Deletes a property |
| `do`         | Control Flow     | Creates a do-while loop |
| `else`       | Control Flow     | Alternative in if |
| `export`     | Module           | Exports a module |
| `extends`    | Class            | Inherits a class |
| `finally`    | Exception        | Runs after try/catch |
| `for`        | Control Flow     | Creates a for loop |
| `function`   | Declaration      | Declares a function |
| `if`         | Control Flow     | Conditional statement |
| `import`     | Module           | Imports a module |
| `in`         | Operator         | Checks property existence |
| `instanceof` | Operator         | Checks prototype chain |
| `let`        | Variable         | Declares a block-scoped variable |
| `new`        | Operator         | Creates an instance |
| `of`         | Operator         | Iterates over values |
| `return`     | Control Flow     | Returns a value |
| `super`      | Class            | References parent class |
| `switch`     | Control Flow     | Multi-way branch |
| `this`       | Context          | Refers to current context |
| `throw`      | Exception        | Throws an error |
| `try`        | Exception        | Attempts a block |
| `typeof`     | Operator         | Returns type as string |
| `var`        | Variable         | Declares a function-scoped variable |
| `void`       | Operator         | Evaluates expression returning undefined |
| `while`      | Control Flow     | Creates a while loop |
| `with`       | Scope            | Extends scope chain |
| `yield`      | Generator        | Pauses/resumes generator |
| `async`      | Async            | Declares async function |
| `await`      | Async            | Waits for promise |
| `static`     | Class            | Declares static method/property |
| `get`        | Class            | Defines a getter |
| `set`        | Class            | Defines a setter |
| `enum`       | Reserved         | Reserved for future use |
